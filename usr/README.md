# 科技創新管理：進路時機個案分析 (Tech Innovation Cases)

這是一個以「科技創新管理」課程中「進路時機 (Timing of Entry)」為主題的互動式個案研討應用程式。本專案彙整了 Uber 與空中交通 (Urban Air Mobility) 領域的四個經典個案，幫助使用者理解創新技術導入市場的時機選擇與其帶來的影響。

## 收錄個案

1. **Uber 的崛起**：探討破壞式創新與完美掌握先行者優勢 (First-mover Advantage) 的商業模式。
2. **Uber Elevate 啟動失敗**：分析在電池技術、基礎設施與法規尚未成熟時，過早進入 (Premature Entry) 市場的教訓。
3. **UberAir 的機會與挑戰**：展望都市空中交通 (UAM) 的未來潛力與仍需克服的現實障礙。
4. **天空之戰**：解析目前 eVTOL 市場的競爭態勢，包含新創公司、傳統航空巨頭與汽車製造商之間的主導設計 (Dominant Design) 之爭。

## 核心功能

*   **跨平台響應式介面**：完美適配手機、平板與桌面環境的閱讀體驗。
*   **Markdown 內容渲染**：清晰、易讀的個案文章排版。
*   **模組化資料結構**：便於未來擴充更多科技創新管理相關個案。

## 技術架構

*   **框架**: Flutter
*   **語言**: Dart
*   **主要套件**:
    *   `flutter_markdown`: 用於渲染高品質的個案分析文本。

## 執行與編譯

在終端機中執行以下指令以啟動應用程式：

```bash
flutter run
```

若要編譯為網頁版本：

```bash
flutter build web
```
